# Niyoggen Website

A polished multi-page Next.js website for Niyoggen, a career, campus, hiring, events, connect, dashboard, and organization ecosystem platform.

## Pages

- `/` Home
- `/careers` Careers mission, steps, and ecosystem
- `/campus` Campus partnerships and student support
- `/connect` Community and mentoring network
- `/events` Career events and programs
- `/hire` Hiring process and employer journey
- `/contact` Contact page
- `/dashboard` Dashboard preview
- `/about` About us

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Build

```bash
npm run build
npm start
```
